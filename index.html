HyperText Markup Language


Usado na construção de páginas da web, a linguagem de marcação HyperText Markup Language, também conhecida pela sigla HTML, são documentos que podem ser interpretados por navegadores.

O físico britânico Tim Berners-Lee foi o criador do HTML. Todo documento HTML utiliza marcadores (do inglês tags), palavras entre parêntes angulares (< >), que são comandos de formatação da imagem.

Também podem ser usados hipertext, que é fazer ligação com outras páginas.


Alguns códigos utilizados


  • <h1>, <h2>, ... <h6>: Títulos que variam de tamanho dentro das prioridades (sua aparência pode ser alterada com CSS - Cascade Style Sheet - Folhas de Estilo em Cascata).
  • <p>: Parágrafo.
  • <br>: quebra de linha.
  • <b>, <i>, <u> e <s>: negrito, itálico, sublinhado e riscado, respectivamente.
